# 🐍 Personal Portfolio — machinatororis.github.io

Welcome to my personal portfolio website, built with **React**, **TypeScript**, and deployed via **GitHub Pages**.

I'm a **Python developer** with experience in building chatbots, integrating machine learning, web development, and supporting project infrastructure.  
My projects include NLP-based assistants, feedback analysis systems, Telegram bots, and various web tools — more coming soon.

## 🌐 Live Website

🔗 [machinatororis.github.io](https://machinatororis.github.io)

## ⚙️ Tech Stack

- ⚛️ React + TypeScript
- 🎨 Sass, CSS Modules
- ⚙️ GitHub Pages (via gh-pages)
- 🐍 Backend experience: Python, Django, RASA, REST APIs

## 🚀 Deployment

To build and deploy this project:

```bash
npm install
npm run build
npm run deploy
```

Website is deployed via GitHub Pages and served from the `gh-pages` branch.
